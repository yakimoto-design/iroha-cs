# 管理画面へのアクセス方法

## 📍 管理画面のURL

サイトを公開した後、以下のURLで管理画面にアクセスできます：

### 方法1: 管理メニューから（推奨）
```
https://あなたのドメイン.com/admin-menu.html
```
このページから「コラム管理」または「サービス管理」を選択できます。

### 方法2: 直接アクセス
**コラム管理:**
```
https://あなたのドメイン.com/admin.html
```

**サービス管理:**
```
https://あなたのドメイン.com/services-admin.html
```

## 🔧 ローカル環境での確認方法

開発環境（まだ公開前）で確認する場合：

### 簡易サーバーの起動方法

#### Python を使う場合（推奨）
プロジェクトフォルダで以下を実行：
```bash
# Python 3の場合
python -m http.server 8000

# Python 2の場合
python -m SimpleHTTPServer 8000
```

その後、ブラウザで以下にアクセス：
```
http://localhost:8000/admin-menu.html
```

#### Node.js を使う場合
```bash
# http-serverをインストール（初回のみ）
npm install -g http-server

# サーバー起動
http-server
```

その後、ブラウザで：
```
http://localhost:8080/admin-menu.html
```

#### VS Codeの拡張機能を使う場合
1. VS Codeで「Live Server」拡張機能をインストール
2. index.htmlを右クリック
3. 「Open with Live Server」を選択
4. URLを `http://localhost:5500/admin-menu.html` に変更

## ⚠️ よくある問題と解決方法

### 問題1: ファイルをダブルクリックで開いている
**症状:** URLが `file:///...` から始まる  
**原因:** ファイルを直接開くと、JavaScriptのfetch APIが動作しません  
**解決:** 上記の簡易サーバーを使用してください

### 問題2: 404エラーが出る
**症状:** ページが見つかりません  
**原因:** URLが間違っているか、ファイルがアップロードされていない  
**解決:** 
- URLのスペルを確認
- ファイルが正しくアップロードされているか確認

### 問題3: 白い画面のまま
**症状:** 何も表示されない  
**原因:** JavaScriptエラーまたはCSSが読み込めていない  
**解決:**
- ブラウザの開発者ツール（F12）でエラーを確認
- CSSファイルのパスを確認

### 問題4: データが表示されない
**症状:** 管理画面は開くが、コラムやサービスが表示されない  
**原因:** API（tables/...）にアクセスできない  
**解決:** 
- Publishタブで公開してください
- 開発環境では制限がある場合があります

## 📱 公開後の推奨ブックマーク

公開したら、以下をブラウザにブックマークしておくと便利です：

```
名前: iroha 管理メニュー
URL: https://あなたのドメイン.com/admin-menu.html
```

## 🆘 それでも開けない場合

以下の情報を確認してください：

1. **現在の状況:**
   - [ ] 公開済み / [ ] まだ公開前
   
2. **アクセス方法:**
   - [ ] ファイルをダブルクリック
   - [ ] ローカルサーバー経由
   - [ ] 公開URLから
   
3. **エラー内容:**
   - [ ] 404エラー（ページが見つからない）
   - [ ] 白い画面
   - [ ] その他: ___________

4. **ブラウザのコンソール:**
   - F12を押して「Console」タブを確認
   - エラーメッセージがあればコピー

これらの情報があれば、より具体的なサポートができます！
